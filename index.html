Build a simple tile server for Esri shapefiles to show them in PTV Ajax Maps

Description

The tutorial demonstrates:

_
  • Implementing a custom tile server with ASP.NET, so you can request a tile image using an url like
    ShapeTileHandler.ashx?x=4&y=2&z=3&layer=someLayer&style=someStyle
  • Configuring ajaxmaps to display the custom tile layer like in the sample below.
  • Utilizing SharpMap from http://sharpmap.codeplex.com for on-the-fly rendering of images from a shape file

You can also use the sample as blueprint to render your custom data format and embed it into Ajax Maps.
